## ü§ù Contributing

Want to add repositories to the library? Thank you!

### Steps:

1. Fork this repository  
2. Add your repo as a string in the `repos.json` file  
3. Commit your changes  
4. Open a Pull Request  

### Example Entry

```json
{
  "url": "https://example.com/repo.json"
}```
